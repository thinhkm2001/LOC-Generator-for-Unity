A Unity Editor tool designed to analyze and visualize the size of your codebase.
It scans .cs files within configurable include/ignore folders, calculates lines of code (LOC), and presents results directly in a sortable editor window.

The tool also supports exporting reports for further analysis, making it useful for refactoring, code reviews, and tracking project growth over time.
